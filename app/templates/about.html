About Mini Twitter

Welcome to Mini Twitter 🎉 — a lightweight social platform built with Flask and SQLAlchemy. This project is a fun experiment to learn the basics of full-stack development including:

  • 🔹 User authentication (login, register, logout)
  • 🔹 Posting and viewing tweets
  • 🔹 Categories for organizing tweets
  • 🔹 Like and view counters
  • 🔹 Clean, simple UI

The goal is to practice building a mini social media app while learning how databases, templates, and routes come together in Flask.

🚀 Made with ❤️ and Flask — for learning and fun.
